# 📈 Büyük Veri Analizi Temelleri

Bu depo, karmaşık ve büyük veri setlerini işleme, temizleme ve bu verilerden stratejik anlamlar çıkarma süreçlerini içerir. Veri analitiğinin "mutfak" kısmı olan veri hazırlama aşamasına odaklanır.

### 🛠️ Neler Var?
- **Veri Temizleme:** Eksik, hatalı ve aykırı değerlerin (outliers) tespiti ve düzeltilmesi.
- **Keşifçi Veri Analizi (EDA):** Veri setinin istatistiksel özetleri ve gizli trendlerin tespiti.
- **Özellik Mühendisliği:** Ham veriden yeni ve anlamlı analiz değişkenleri türetme.

### 🚀 Kullanılan Teknolojiler
- **Dil:** Python
- **Kütüphaneler:** Pandas, NumPy
- **Yöntemler:** Veri Normalizasyonu, İstatistiksel Analiz
